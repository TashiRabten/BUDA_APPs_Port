TradutorBUDA

Java-based Tibetan translation and text-processing application focused on Buddhist terminology workflows.

Tech stack: Java 17, Maven, SQLite, ONNX Runtime, tokenizers, R/Rserve, OCR utilities.

  • Tibetan term lookup and processing workflows
  • Word-breaking and Tibetan text utilities
  • Model/ranking and OCR-related optional integrations

GlossarioBUDA (Simplified)

Simplified glossary application retaining the core term management features without experimental OCR integration.

Tech stack: Java, SQLite, SWT, XML processing.

  • Add, edit, delete, and search Buddhist terms
  • Supports Tibetan, Sanskrit, Pali, English, Portuguese
  • XML import, WhatsApp-friendly export, always-on-top quick view
