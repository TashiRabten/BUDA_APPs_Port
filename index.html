TradutorBUDA

Java-based Tibetan translation and text-processing application focused on Buddhist terminology workflows. It combines dictionary lookup, tokenization, transliteration, ranking/model services, and optional OCR-related utilities.

Tech stack: Java 17, Maven, SQLite, ONNX Runtime, tokenizers, R/Rserve, OCR utilities

  • Tibetan term lookup and processing
  • Word-breaking and Tibetan text utilities
  • Optional OCR-related tools and assets

GlossarioBUDA - Simplified Version

Simplified version of GlossarioBUDA containing the core glossary management functionality without the experimental Namsel OCR integration.

Tech stack: Java, SQLite, SWT, XML processing

  • Add, edit, delete, and search Buddhist terms
  • Supports Tibetan, Sanskrit, Pali, English, and Portuguese
  • XML import, WhatsApp-friendly export, and compact always-on-top view
